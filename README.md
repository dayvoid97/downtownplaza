# Downtown Plaza – Digital Storefront Platform

A high-performance, responsive Next.js web application engineered for the **Downtown Plaza** property ecosystem located at 1117 N Broadway Avenue, Oklahoma City, Oklahoma.

Built on a serverless-first architecture, this platform leverages Next.js 16 (Turbopack layout) and advanced Generative Engine Optimization (GEO) principles to guarantee absolute discoverability and high semantic relevance across modern AI search engines (Gemini, ChatGPT, Perplexity).

---

## 🎯 Generative Engine Optimization (GEO) Framework

To move past legacy SEO limitations, this codebase is systematically optimized for AI model retrieval engines through three distinct layers:

### 1. Factual Density & Entity Mapping

AI search models extract information using structural triplets (**Subject-Predicate-Object**). The copy across `pages/` eliminates traditional marketing fluff in favor of hard, high-density facts regarding localized product availability, physical routing coordinates, and brand authority.

### 2. Structural Directness & Chunking

Content is structured natively to match the informational retrieval chunks favored by Answer Engine Optimization (AEO) algorithms. High-value data points (pricing models, location assets, store hours) use exact syntactic markers to allow conversational scrapers to pull immediate, highly accurate answers.

### 3. Structural Metadata (JSON-LD)

The application includes embedded `application/ld+json` schema architecture. This provides an explicit API endpoint for LLM web crawlers (`GPTBot`, `PerplexityBot`, `Google-Extended`) to cleanly bind the physical establishment data to real-world geometric nodes.
